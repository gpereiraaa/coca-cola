O que é um repositório no Git e qual a sua importância?
R: É basicamente um diretório com arquivos só que hospedado na nuvem, podendo ser acessado de qualquer local, ele é muito importante pois ele armazena todas as etapas de um projeto, também mais de uma pessoa pode trabalhar no projeto ao mesmo tempo facilitando muito. 

Qual a função do Git Bash?
R: Ele é o aplicativo que entende os comandos de Git assim facilitando para usar quando precisar subir um diretorio no git.

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: O git add ele serve para adicionar quais arquivos serão enviados para o próximo commit, o git add . serve para adicionar todos os arquivos que estão no diretório já o outro serve para adicionar um determinado arquivo.

Por que utilizamos o ponto (.) no comando git add .?
R: Para que ele adicione todos os arquivos.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R: Ele é o comando que adiciona uma mensagem no proximo arquivo que será enviado para o GitHub, é importante pois a pessoa saberá sempre o que mudou nesse arquivo.

Qual a função do comando git push?
R: Ele é o comando que puxa clona o arquivo que está no seu computador e envia para o GitHub.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R: Pois caso a pessoa queira procurar algo que tenha feito antes no projeto ela encontrará e ainda com uma breve descrição.
